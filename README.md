# Test task sockets
Socket server which accept http requests 
***
## Features
- Create rooms
- Connect to room by room id
- Send message to room
***
## Technology stack
- Python 3.10.6
- python-socketio 5.8.0
- faker 19.1.0
- Pydantic 2.0.3
- Poetry 1.4.1
- aiohttp 3.8.4

***
## Start app
1. clone app
   ```
   git clone https://github.com/AndrewIsaev/test_task_sockets.git
2. Install dependencies
   ```
    poetry install
3. Run app
   ```
    python3 main.py
```
